BBSlink is a free InterBBS games server that allows any SysOp to easily add registered door games to their system
https://bbslink.net/

Requires: curl

 1. Register on https://bbslink.net/sysop.php and wait for configuration
    information to be emailed to you.
 2. Copy vars.sh.example to vars.sh
 3. Edit vars.sh with your configuration data
 4. Enable in BinktermPHP Admin -> Doors -> Native Doors
