# Overview

This is the matterbridge install directory for BinktermPHP.

# Install 

Visit https://github.com/42wim/matterbridge/releases and download the latest
or preferred release of matterbridge for your platform.  Rename it to
"matterbridge" and make it executable, eg:

```bash
mv matterbridge-1.26.0-linux-64bit matterbridge
chmod +x matterbridge
```
