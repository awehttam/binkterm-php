# Reverse Polarity Webdoor

You can use this webdoor to connect to the Reverse Polarity BBS system.  It can also be used as a reference for creating
your own web door to connect to your text based bulletin board.

- Copy revpol to myboardname
- Edit webdoor.json to reflect the new id, description, etc
- Update /webdoors/revpol/ to /webdoors/myboardname within the files of myboardname/
- Add to and enable within config/webdoors.jsonn
