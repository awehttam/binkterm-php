TW2002 V3 
Downloadable from: https://www.wiki.classictw.com/index.php?title=Application:TradeWars_2002_v3

Copy the downloaded file into your TW2002 dos DOORS directory and run the installer.  This will extract the files.

Run TEDIT to configure your nodes.  You must configure each node # individually and the number you need is the number
of simultaneous players playing the game at the same time.

For each node configuration you'll need at least:

 Drop file type: DOOR.SYS
 Drop file location:  C:\DROPS\NODEx
 
These settings get the game up and running as far as connecting to the game goes.
