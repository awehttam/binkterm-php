Subj: BinktermPHP 1.8.7 Preview - What's Coming

Hello everyone,

I wanted to share a preview of what's coming in BinktermPHP 1.8.7.  This
has been a big development cycle and I'm pretty excited about where the
project is heading.  Here's a tour of the highlights.

--- FILE AREAS ---

The file area experience has been completely overhauled.  Clicking a file
now opens an inline preview modal instead of immediately downloading.

Supported preview types include images (lightbox view), video and audio
(native browser playback), text/NFO (with automatic CP437-to-UTF-8
conversion so your old ASCII art looks right), and ANSI art rendered with
the full 16-colour palette.

For you retro computing folks:

  * .prg files are decoded using the C64 screen RAM decoder and displayed
    with the authentic 16-colour C64 palette.  A "Run on C64" button
    appears in the nav bar and loads a full jsc64-based C64 emulator
    inline - no page navigation required.

  * .seq PETSCII stream files are rendered through the PETSCII decoder and
    can also be run live in the emulator.

  * .d64 floppy disk images are parsed, the directory is extracted, and all
    PRG files are shown in a gallery with the disk name as a header.

  * .rip RIPscrip files are now rendered server-side to SVG and displayed
    inline.  RIP files inside ZIP archives are shown as a gallery too.

  * ZIP files now open as a browsable file listing.  You can click any
    previewable entry (image, text, ANSI, RIP, PRG...) to open it inline,
    and every entry has a download button regardless of type.  Old DOS ZIPs
    using implode or shrink compression are handled via a fallback to the
    system unzip utility, with a graceful notice if extraction isn't
    possible.

Navigation arrows and keyboard shortcuts let you move through the file
list without closing the modal.

Also new in file areas:

  * Global file search - a search box in the sidebar searches filename
    and description across all accessible file areas at once.

  * ISO-backed file areas - point a file area at a CD/DVD ISO image or
    any readable directory path.  BinktermPHP imports the directory tree
    using FILES.BBS, DESCRIPT.ION, Simtel-format 00INDEX.TXT, and other
    standard catalogues.  Great for exposing old shareware CD collections.

  * File area comments - link any file area to an echomail echo area and
    users can leave threaded comments on individual files directly from
    the preview modal or file detail panel.  Comments are real FTN
    echomail messages with a ^AFILEREF kludge, so other nodes that carry
    the echo see them too.  The echomail reader shows a banner on comment
    messages linking back to the file.  LovlyNet sysops: link your file
    areas to LVLY_FILECHAT.

  * Gemini support - file areas can be published to the Gemini capsule,
    allowing Gemini clients to browse listings and download files.

  * Outbound FREQ support - request files from remote binkp systems using
    the new scripts/freq_getfile.php CLI tool.

--- MESSAGE READER ---

  * Advanced search with per-field filtering (poster, subject, body) and
    date range picker.  Backed by new pg_trgm trigram GIN indexes so it's
    fast even on large databases.

  * Kludge lines are now hidden by default and toggled with a small icon
    button in the toolbar.

  * A print button opens the message in a clean popup window.

  * FTN node addresses in From:/To: fields are clickable links to the
    nodelist node view.

  * The reader now transparently loads the next page when you reach the
    end of one.  When you finish the last message in an echo, it offers
    to jump to the next subscribed echo with unread messages.

  * Page position memory - the reader remembers which page you were on
    per echo area and restores it when you return.  Opt-in in Settings.

  * RIPscrip messages in echomail are detected and rendered inline.

--- LOVLYNET ---

A new admin page (Admin -> Area Management -> LovlyNet Areas) shows all
echo and file areas available on LovlyNet with their subscription status.
Toggle subscriptions with a single click.  Credentials are read from
config/lovlynet.json automatically.

--- NODELIST ---

  * New interactive Leaflet map tab with network colour coding and marker
    clustering.  Nodes are geocoded from their location field.

  * Nodelist search now has a multi-select flag filter (CM, IBN, INA,
    FREQ, etc.) with plain-English descriptions.

--- OTHER HIGHLIGHTS ---

  * Netmail forwarding to email - users can opt in to have incoming
    netmails forwarded to their email address, including attachments.

  * Echomail digest - subscribe to receive a periodic digest of echo area
    messages by email.

  * Telnet/SSH file areas now support virtual subfolders, show all areas
    (not just the first page), and ISO-backed areas can be downloaded
    via ZMODEM.

  * Database Statistics admin page showing size, growth, query
    performance, index health, and maintenance status.

  * Admin menu reorganized with Analytics and Community submenus.

--- REGISTRATION ---

1.8.7 introduces a registration system as a way for sysops to support
continued development of BinktermPHP.  Registration is completely
optional - the software is fully functional without it and always will
be.  Registered installations get a small set of extras as a thank-you
for contributing.

Premium features in 1.8.7:

  * Custom branding - set your own footer text and optionally hide the
    "Powered by BinktermPHP" attribution line.

  * Message templates - save and reuse subject/body pairs in the compose
    form, filterable by message type (netmail, echomail, or both).

  * Custom splash pages - configure Markdown content that appears above
    the login and registration forms.  Good for announcements or a
    welcome message specific to your BBS.

  * Netmail forwarding to email - users can opt in to have incoming
    netmails forwarded to their email address, including attachments.

  * Referral analytics - see who has referred the most users, recent
    sign-ups, and bonus credits earned through referrals.

  * Economy Viewer - the credit economy dashboard is now a registered
    feature.

To register, visit Admin -> Licensing and click "How to Register" for
current options.

--- UPGRADING ---

Full upgrade notes including migration details, configuration changes,
and step-by-step instructions are at:

  https://github.com/awehttam/binkterm-php/blob/claudesbbs/docs/UPGRADING_1.8.7.md

As always, back up your database and files before upgrading.  The 1.8.7
migration rebuilds trigram indexes on the message tables which may take
a few minutes on large installs - don't interrupt it.

Thanks for running BinktermPHP.  Feedback and bug reports welcome.
